## Build Application 
1- Clone the repository: 
git clone https://github.com/gcmgomes/pace2023.git

2- Create a build directory: 
mkdir build && cd build

3- Run cmake: cmake

4- Run make: make

## Run Tests
To run the tests, pass the name of a test file, as shown below (from build folder):
1- ./pace < tests
